# Student-Stress-Level-Predictor
College students experience academic stress due to workload, deadlines, exams, and sleep issues. Early detection of stress levels can help students take corrective steps. This project predicts whether a student’s stress level is:  Low  Medium  High based on simple features like:  Sleep hours Study hours  Number of assignments etc.
 Student Stress Level Prediction System

Predicts student stress levels using:
- Rule-based reasoning
- BFS search
- Conditional probability
- Decision Tree ML model

## Features
- User input module
- Rule-based inference
- ML classifier
- Probability estimator
